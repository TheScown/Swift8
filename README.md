# Swift8 

A macOS Chip 8 emulator written in Swift.

Requires macOS High Sierra or later.

Features include a graphical debugger and touch bar controls (where available). 

## Key Mapping

Swift8 uses the following keyboard mapping:

| Mac | Swift8 |
| :---: | :---: |
| 1 | 1 |
| 2 | 2 |
| 3 | 3 |
| 4 | c |
| q | 4 |
| w | 5 |
| e | 6 |
| r | d |
| a | 7 |
| s | 8 |
| d | 9 |
| f | e |
| z | a |
| x | 0 |
| c | b |
| v | f |
